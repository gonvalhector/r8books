About

R8Books is a web application written in Python 3.6.9 that uses the Django 3.0.8 web framework and is currently being hosted on the Heroku platform. You can find the GitHub repository here.

Originally designed with the Flask framework for the CS50’s Web Programming with Python and JavaScript course.

API

Authenticated users can make a GET request to the /api/ route, where is an ISBN number. R8Books will return a JSON response containing the book’s title, author, publication date, ISBN number, review count, and average score.
